CVR View

Easily host code coverage reports for your LCOV, Cobertura, Jacoco, or Go Cover results.

CVR View is an Express application for hosting code coverage results.

If you'd like to host your coverage internally, or without paying for monthly services, CVR View might be for you.

  • Integrates with GitHub repos and pull request statuses
  • Works with your choice of CI server
  • Easy to setup
  • Open source and MIT licensed

Check out the seed project to get started and make future updaties as simple as running npm install.

Feature Tour

Activate any repos in your GitHub account

Browse files from recent commits
